##### 2.1.1 变量的命名和使用
- 变量名只能包含字母、数字和下划线。（只可用字母或下划线开头）
- 变量名不可包含空格
- 不要将Python关键字和函数名用作变量名
- 变量名应简短又具有描述性
- 慎用小写字母l和大写字母O
